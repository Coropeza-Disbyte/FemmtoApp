# Release Summary — v3.9.0

> Versión con un único cambio funcional respecto a v3.8.0: introducción del modal promocional `HotFemmtoModal` en la pantalla Home para potenciar el tráfico de usuarios argentinos al marketplace de Femmto.

---

## Cambios incluidos en esta versión

| # | Feature / Pantalla | Tipo | Descripción |
|---|--------------------|------|-------------|
| 1 | HotFemmtoModal | Nueva pantalla | Modal promocional "Hot Femmto" visible solo para usuarios con `country = 'AR'`, una vez por día, con countdown y CTA al marketplace |

---

## Sin otros cambios

Todas las demás pantallas y features de v3.8.0 se mantienen sin modificaciones.
