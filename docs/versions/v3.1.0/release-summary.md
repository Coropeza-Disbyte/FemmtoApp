# v3.1.0 — Release Summary

**Tipo:** Migración AWS + Nuevas funcionalidades  
**Rama git:** feat/version-3.1.0  
**Baseline del framework:** primera versión documentada con rama disponible

## Cambios de alto impacto

- Migración completa a AWS
- Integración báscula corporal BWS12 — registro de mediciones desde dispositivo
- App habilitada en Ecuador para iOS
- Opción "Eliminar cuenta" en Mi cuenta (cumplimiento legal Android e iOS)
- Ventanas de mantenimiento — restringe uso de app durante mantenimiento
- Actualización requerida — bloquea acceso hasta que el usuario actualice
- Modal de actualización opcional

## Bottom Navigation en esta versión

5 tabs: Home · Compartir · Medición · Dispositivos · Recordatorios
