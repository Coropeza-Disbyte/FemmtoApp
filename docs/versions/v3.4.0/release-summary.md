# Release Summary — v3.4.0

## Cambios respecto a v3.3.0

| Área | Tipo | Descripción |
|------|------|-------------|
| HealthNativeIntro | Nueva pantalla | Onboarding de integración con Health Connect (Android) / Apple Health (iOS) |
| HeartRateDetails | Nueva métrica | Detalle de Frecuencia Cardíaca con gráficos día/semana/mes |
| HeartRateMeasurementHistory | Nueva pantalla | Historial de mediciones de frecuencia cardíaca |
| StepsDetails | Nueva métrica | Detalle de Pasos con gráficos día/semana/mes |
