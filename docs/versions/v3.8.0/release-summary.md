# v3.8.0 — Release Summary

**Tipo:** Push Notifications Segmentadas  
**Fecha aprox.:** 2024

## Cambios de alto impacto

- Sistema de push notifications segmentadas por usuario, dispositivo y comportamiento
- Pantalla `Notifications` — listado de notificaciones recibidas
- Badge de notificaciones no leídas en el header del Home

## Fix incluido

- OCR para mediciones de presión arterial en Android

## Pantallas afectadas

| Pantalla | Tipo de cambio |
|----------|----------------|
| Home (header) | Badge de notificaciones agregado |
| Notifications | Introducida |
